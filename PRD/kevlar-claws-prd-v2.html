PRD: Kevlar-Claws v2.0 (The "Adoption & DX" Protocol)

1. Executive Summary

Phase 1 established a "Zero-Trust" architectural baseline. Kevlar-Claws v2.0 evolves the tool from a "strict enforcer" to a "high-velocity, trusted co-pilot" by introducing token-efficient context modeling, interactive fallback mechanisms, and local RAG caching.

2. Key Functional Requirements

  • FR-01: Token-Optimized Context (Skeleton AST): Reduces context usage by 60%+ using hierarchical AST summaries.
  • FR-02: Interactive Desync Resolution: Replaces hard aborts with an interactive CLI flow to resolve state conflicts.
  • FR-03: Local RAG Vector Cache: Accelerates codebase retrieval using local embedding models (ChromaDB/FAISS).
  • FR-04: Headless CI/CD Mode: Adds --headless support for automated environment integration.
